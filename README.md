# 🎹 CSS Piano

This project is a visual piano built using HTML and CSS as part of my front-end development learning journey.

The goal of the project was to practice CSS layout, positioning, and responsive design techniques by recreating a piano keyboard using only HTML elements and CSS styling.
## you can see the project here :

## What I Learned

While building this project, I practiced several important CSS concepts:

Box sizing

Using box-sizing: border-box to make element sizing more predictable.

Positioning

Using position: relative and position: absolute to correctly place elements.

Pseudo-elements

Using ::before and ::after to create additional visual elements without adding extra HTML.

Responsive design

Using CSS media queries to adjust the layout for different screen sizes.

CSS layout techniques

Structuring elements to visually recreate piano keys.
